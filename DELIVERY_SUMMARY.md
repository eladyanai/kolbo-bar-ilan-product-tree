## 🎓 HOW TO USE

### For Users
1. Open https://kolbo-bar-ilan-product-tree.vercel.app/
2. Drag items with visible handles (☰)
3. Search by name or Commax ID (#89)
4. Select multiple items with checkboxes
5. Click buttons for bulk actions
6. Save changes with "💾 Save" button

### For Developers
1. Clone: `git clone https://github.com/eladyanai/kolbo-bar-ilan-product-tree.git`
2. Run: `cd product-tree && python3 -m http.server 3000`
3. Edit: Modify `index.html` directly (no build needed)
4. Push: Changes auto-deploy to Vercel

### API Usage
```javascript
// POST to /api/save-data
const data = {
  groups: [
    {
      id: 1,
      name: "Group Name",
      departments: [
        {
          id: 1,
          name: "Department",
          subcategories: [
            {
              id: 1,
              name: "Subcategory",
              commax_id: 89,
              itemCount: 31,
              notes: ""
            }
          ]
        }
      ]
    }
  ]
};

fetch('/api/save-data', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(data)
})
.then(r => r.json())
.then(result => console.log('Saved:', result));
```
