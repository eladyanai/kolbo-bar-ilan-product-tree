🧪 How to Test All Features

1. Drag & Drop
1. Open the dashboard
2. Hover over any item - see drag handle (☰)
3. Click and hold the handle
4. Drag over another item - see blue highlight
5. Release to drop
6. Watch cursor change to grabbing hand
2. Search & Filter
1. Type in search box: "טיגון" or "#89"
2. Results filter instantly
3. Click group color dots in sidebar to filter
4. Use top dropdown to filter by group
3. Bulk Actions
1. Check 2-3 items with checkboxes
2. Action bar appears at top
3. Click "Delete" → confirmation modal
4. Click "Move" → select group and department
5. Click "Notes" → add bulk notes
4. Save to Cloud
1. Make a change (delete, move, or edit)
2. See "⏱️ Unsaved changes" indicator
3. Click "💾 Save to Cloud"
4. See "💾 Saving..." then "✅ Saved to cloud"
5. Check API response (posts to /api/save-data)
5. Responsive Design
1. Open on desktop - full layout
2. Resize to tablet - sidebar adjusts
3. Resize to mobile - sidebar becomes horizontal
4. All controls stay accessible
